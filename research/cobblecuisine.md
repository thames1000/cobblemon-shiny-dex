# CobbleCuisine — items, recipes & effects

Source: CobbleCuisine mod (github.com/Fyre520/CobbleCuisine-Java) data + Java enums, with Cobbleverse v29 datapack recipe overrides applied. Spawn/catch/EV/shiny buffs last **6000 ticks (5 min)** by default (config-tunable); Beans/Malasadas/Poké Puffs give vanilla **Regeneration** (10s/30s/15s). Recipes are shapeless crafting unless tagged **[shaped]**; "(any)" = a tag (any matching item); "N×" = quantity; "__OR__" = alternate recipes; *(Cobbleverse-overridden)* = v29 datapack replaces the mod default.
